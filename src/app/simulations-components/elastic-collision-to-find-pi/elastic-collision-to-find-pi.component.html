There are tons and tons of different ways to find the digits of PI. And they are also way more efficient.

But there is also this way where we make 2 blocks collide with each other (without any loss of energy, i.e., elastic collision) and if we count the number of collisions, then it surprisingly resembles the digits of PI. The relation between the blocks is that if 1 block weighs x, then the other should weigh (100 precision - 1) times x where the precision is the number of digits or the accuracy, we desire in the number of collisions.

It is fascinating to see such a connection between PI and simple physics. If you are interested in learning more about why this works, then I would highly recommend that you check this 3Blue1Brown's video where he goes in to the details of this.
